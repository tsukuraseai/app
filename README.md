# Shield Breaker - Overdrive Mod

## 改造内容（概要）
- **新パワーアップドロップ**を追加しました（Widen / Multiball / Overdrive / Repair）。敵撃破時に一定確率で落下し、シールドで回収できます。
- **FEVERゲージ & FEVER MODE**を追加しました。敵撃破・パーフェクト反射でゲージ上昇、満タンで一定時間スコアと殲滅力が上がります。
- **コンボシステム**を追加しました。連続撃破でコンボが伸び、得点ボーナス・演出テキストが出ます。
- **Elite敵**を追加しました。高耐久・強化弾で後半ウェーブの圧力を上げています。
- **視覚演出を強化**しました（ボール残像、オーバードライブ時の発光、FEVER画面オーラ、UI表示強化）。
- **タイトル画面とキャンバス外観**を微調整し、全体のSF感を向上しました。

## 【必要素材リスト】
- 追加画像素材は不要（既存素材＋Canvasエフェクトのみで実装）。

## 新規SEが必要な場合（候補ファイル名）
- 追加SEは必須ではありません（既存SEのみで動作）。
- もし強化する場合の候補:
  - `se_powerup_pick.mp3`（パワーアップ取得音）
  - `se_fever_start.mp3`（FEVER開始音）
  - `se_combo_up.mp3`（コンボ節目到達音）
